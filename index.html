Программы / Windows / LogViewer







  Просмотр длинных логов. Особенность программы в загрузке небольших фрагментов файла, просматриваемых в данный момент, а не всего файла целиком. Это позволяет быстро открывать очень длинные лог-файлы. Есть возможность захвата строк или страниц в отдельный файл. Можно выделить строки с определённым текстом. Программа написана во время администрирования собственного сервера, из-за маломощного рабочего компьютера с малым объёмом памяти.

Утилита портирована на Linux. Скачать версию для Ubuntu 18.04 LTS. Исходник прилагается - из него можно попробовать собрать программу под другие версии Unix. Для выполнения программы необходимо выполнить команду "chmod ugo+x lv" или установить атрибуты исполняемого файла в свойствах (правый клик мышью на файле).
