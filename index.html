LogViewer

 Application for log viewing.

 Scroll keys: NumPad / left, right, up, down - arrows,
              Home, End, (Shift+ / Ctrl+) PgUp, (Shift+ / Ctrl+) PgDown
              (Shift allows to jump quickly to any part of log).
        Ins - highlight a line.
      Enter - store a page or line into "LogViewer.txt"
              (appears in the viewed file folder).
        Esc - quit.

 Program reads only a viewed part of file, so it takes a minimum of
 system memory. Maximum size of log is near 2 Gb. If file's bigger,
 it will still opens, and accessible persentage of one will be shown at
 the status line (in brackets).

 Tested on Windows XP, 10.

 To open a file with the utility drop it on LogViewer's icon or window.
 Also file can be opened with right mouse click and choice of this application.
 It's may be convenient to mark the checkbox "open with this program" for files
 with ".log" extension.
